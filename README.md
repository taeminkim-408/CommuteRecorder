### 대략적인 기능 설명   
1. 출근
  -인부의 상태를 출근 처리한다. 
2. 퇴근
  -인부의 상태를 퇴근 처리한다.
3. 야근
  -야근 처리를 한다.
4. 조퇴
  -퇴근이 아닌 조퇴처리를 한다.
5. 4대 보험 신청
  -각 인부의 4대 보험을 신청 신청하게 된다면 어떤 이유로 신청하는지 기재한다.
6. 인부 교체
  인부를 다른 인부로 교체한다.
7. 파일 저장
  인부의 명단과 명부를 txt파일로 저장한다.
8. 명부 
  인부의 명단과 출퇴근 상태 4대 보험 신청에 대해서 표로 나타낸다.
9. 인부 추가 
  -공사장 인부를 임름 을 지정하여서 추가한다.
10. 인부 제거
  - 공사장 인부를 지정하여서 제거한다.
